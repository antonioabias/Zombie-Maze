Zombie Maze started as a desktop app I made with my team for our Senior High School Software Festival back in 2017. The original Java code is gone, lost to a corrupted hard drive. So I rebuilt it from scratch for the web and called it Zombie Maze Reimagined.

Same core idea: navigate a maze, avoid zombies, find the exit. New everything else: browser-based, new engine, new visuals, way better AI.
Built with: Phaser.js, JavaScript

Play it: zombie-maze-demo.vercel.app
